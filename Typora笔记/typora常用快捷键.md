# typora常用快捷键

## 文件

|        描述         | 快捷键           |
| :-----------------: | ---------------- |
|    快速打开文件     | ctrl + p         |
|     选中当前行      | ctrl + L         |
| **跳转到文首/文末** | CTRL + home/end  |
|       公式块        | CTRL + shift + m |
|      **引用**       | CTRL + shift + q |





##    段落

| 描述               | 快捷键           |
| :----------------- | ---------------- |
| 增大/减小 标题级别 | CTRL + -=        |
| **段落/标题**      | CTRL +0/12345    |
| **表格**           | CTRL + t         |
| 代码块             | CTRL + shit + k  |
| 公式块             | CTRL + shift + m |
| 引用               | CTRL + shift + q |
| **有序列表：**     | Ctrl+Shift+[     |
| **无序列表：**     | Ctrl+Shift+]     |

## 格式

| 描述         | 快捷键           |
| ------------ | ---------------- |
| **加粗**     | CTRL + b         |
| 斜体         | CTRL + i         |
| 超链接       | CTRL + k         |
| **图片**     | CTRL + shift + i |
| **清除样式** | CTRL + \         |

## 视图

| 描述              | 快捷键           |
| ----------------- | ---------------- |
| 显示/ 隐藏 侧边栏 | CTRL + shift + L |
| 大纲视图          | CTRL + shift + 1 |
| 文件树视图        | CTRL + shift + 3 |
| 文档列表视图      | CTRL + shift + 2 |
| 专注模式          | F8               |
| 全屏 / 退出全屏   | F11              |
| 放大              | CTRL + shift + = |
| 缩小              | CTRL + shift + - |
| typora文件切换    | CTRL + tab       |
